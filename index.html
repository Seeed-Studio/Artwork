Arch Pro

Arch Pro is a variant of mbed LPC1768 with build-in Ethernet, USB Host/Device, Grove connectors and Arduino form factor. With a variety of Shield and Grove modules and lots of software libraries for Arch Pro, you can implement Ethernet, USB Host/Device and NFC applications rapidly and easily.

Pinout

There is a monochrome version of Arch Pro's pinout for you to print and read without a computer.

Fritzing part

If you enjoy using fritzing, there is a part of Arch Pro for you.
